To solve Kepler problem, we need to solve \begin{equation}\label{eq:ode_2nd} \frac{\mathrm{d}^2 \zeta}{\mathrm{d}t^2} = \frac{1}{\zeta^3} - \frac{1}{\zeta^2} \end{equation} with initial condition \begin{align*} \zeta(t_0) &= \zeta_0 \\ \zeta\textnormal{\textquotesingle}(t_0) &= \zeta\textnormal{\textquotesingle}_0 \end{align*} To solve above 2nd order ordinary differential equation  \eqref{eq:ode_2nd}  numerically, we need to approximate 2nd derivative as finite difference. Suppose that the solution $\zeta(t)$ has ~\newline
 continuous 4th order derivative in the Domain $[t_0, t_f]$. then \begin{equation}\label{eq:4thtaylor} \zeta(x+h) = \zeta(x) + \zeta\textnormal{\textquotesingle}(x)h + \frac{1}{2!}\zeta\textnormal{\textquotesingle}\textnormal{\textquotesingle}(x)h^2 + \frac{1}{3!}\zeta\textnormal{\textquotesingle}\textnormal{\textquotesingle}\textnormal{\textquotesingle}(x)h^3 + \frac{1}{4!}\zeta^{(4)}(\eta)h^4 \end{equation} for some $\eta(x, h) \in (t_0, t_f)$. Using 4th order taylor approximation  \eqref{eq:4thtaylor} , we can get following equation \begin{equation}\label{eq:finite_diff} \zeta(x-h)-2\zeta(x)+\zeta(x+h) = h^2\zeta\textnormal{\textquotesingle}\textnormal{\textquotesingle}(x) + O(h^4) \end{equation} Next uniformly divide the domain $[t_0, t_f]$ into $ n $ sub intervals. let $ x_i $ be the end points of the sub intervals then for $ 0 \leq i \leq n $, \begin{equation}\label{eq:x} x_i = t_0 + ih \end{equation} , where $ h = (t_f-t_0)/n $. Now for $ 0 \leq i \leq n $, define $\zeta_i $ as following \begin{equation}\label{eq:zeta_i} \zeta_i = \zeta(x_i)\end{equation} Then we can rewrite finite difference equation  \eqref{eq:finite_diff}  as following \begin{equation}\label{eq:revised_fdiff} \zeta_{i-1}-2\zeta_i+\zeta_{i+1} = h^2\zeta\textnormal{\textquotesingle}\textnormal{\textquotesingle}_i + O(h^4) \end{equation} for $ 1 \leq i \leq n-1 $.
