Solve Kepler problem via finite difference Method \hypertarget{index_req}{}\doxysection{Requirements}\label{index_req}
To install this program, you should have
\begin{DoxyItemize}
\item C++ compiler like g++
\item gnu make 
\end{DoxyItemize}\hypertarget{index_install}{}\doxysection{Installation}\label{index_install}
Type make, then we can see hw1 executable file in bin directory \hypertarget{index_hwoto}{}\doxysection{How To Use}\label{index_hwoto}
Execute hw1 then, it will interactively read
\begin{DoxyItemize}
\item inital condition
\item number of gird points to evaluate
\item output file name
\end{DoxyItemize}
