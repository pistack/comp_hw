By conservation of angular momentum, Angle $ \theta $ satisfies following relation \begin{equation}\label{eq:theta_ode} \frac{\mathrm{d} \theta}{\mathrm{d} t} = \frac{1}{\zeta^2} \end{equation} Integrate both side then we can deduce \begin{equation}\label{eq:theta_int} \theta(t) = \theta_0 + \int_{t_0}^t \frac{1}{\zeta^2} \mathrm{d} t \end{equation} Let $ \theta_i = \theta(t_i) $ as in \mbox{\hyperlink{fdm}{Finite difference Method}}, then for $ 1 \leq i $, \begin{equation}\label{eq:theta_recurr} \theta_i = \theta_{i-1} + \int_{t_{i-1}}^{t_i} \frac{1}{\zeta^2} \mathrm{d} t \end{equation} Next approximate the integral using trapezoidal rule then \begin{equation}\label{eq:theta_approx} \theta_i \approx \theta_{i-1} + \frac{t_i-t_{i-1}}{2}\left(\frac{1}{\zeta_{i-1}^2}+\frac{1}{\zeta_i^2}\right) \end{equation} $\theta_i $ has $ O(n^{-3}) $ local turncation error for trapezoidal rule and additional $ O(n^{-3}) $ for the global turncation error of $ \zeta $ (see \mbox{\hyperlink{fdm}{Finite difference Method}} accurcay).
